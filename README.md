# tick_molecular_protocols

Protocols used for Ixodes scapularis (black-legged ticks) DNA extractions, gel electrophoresis, bacterial 16S rRNA PCR amplification (full-length and V4), library preparation for Illumina MiSeq and Nanopore Sequencing at Colautti lab, Queen's University, Canada. All these protocols were modified with help of Suman Tapar Magar, Heidi Albers, Jaylen Adamson, Damian Bourne, Dr. Sima Afshernezhad and Dr. Zhengxin Sun.

Source of initial protocol 

16S - V4 regions - EMP 16S Illumina Amplicon Protocol V.2, EMP Consortium - https://dx.doi.org/10.17504/protocols.io.kqdg3dzzl25z/v2
