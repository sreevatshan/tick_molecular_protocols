# tick_molecular_protocols

Protocols used for Ixodes scapularis (black-legged ticks) DNA extractions, bacterial 16S rRNA PCR amplification (full-length and V4), library preparation for Illumina MiSeq and Nanopore Sequencing at Colautti lab, Queen's University, Canada. All these protocols were co-developed with help of Suman Tapar Magar, Heidi Albers, Jaylen Adamson, Damian Bourne, Dr. Sima Afshernezhad and Dr. Zhengxin Sun.
